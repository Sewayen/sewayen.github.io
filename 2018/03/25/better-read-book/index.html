《更好的读书课》笔记

发表于 2018-03-25 | 分类于 读书 |   |   字数统计: 1,555(字)   |   阅读时长: 5(分)

社群中大多数伙伴是恒星，自身想要爆发自己的能力，也决定了社群会发出什么样的光芒。
部分伙伴是行星，在恒星伙伴的影响下也能发光。
还有少部分伙伴像是脱落，要靠小皮鞭的抽打才有转动的模样
极个别的可能暂时迷失了方向，而我们，始终无法叫醒一个装睡的人

——引自：剽悍读书营开营仪式 - 《更好的读书课》序言


前言

读书课：理论、方法
开营仪式：具体实践方式
以上需要结合学习
读书：有价值，有意义的事儿，总结的很多方法被束之高阁

更好的读书课

警惕快速阅读

遇到“快速”二字，需要警惕！
建房子，需要打地基，同样，学习一定要经过漫长的过程，才会有大的收获。
例如炒菜，不是买几本食谱就能成为高手，而是经过观摩、大量实践后，才能做出好吃的菜。
经过大量阅读，会有词汇练习、阅读练习，读书速度会明显上升，而不是通过快速阅读的方法。
读书：通过大量的阅读练习，积累词汇量，大脑处理信息能力变强，这样理解的质量和速度才会有明显提升，阅读速度自然上去了。

如何提升选书能力

俗话：选择比努力更重要
想要提升输出质量，首选要选对材料
很多时候到书店选到不好的书，很多时候是：书店逛少了
剽悍一只猫如何选书：逛书店是最好的娱乐之一（一年逛上百次书店）。在书店翻多了，选择能力自然提升上去了。翻着翻着就能发现这本书不错，或这本书不太好。快速在书架上瞄，快速判断&买下。
如何提升选书能力：参考豆瓣优质书评；自己去书店多翻书，选书能力自然提升

好的阅读习惯

刚开始拾起看书时，容易犯困、没劲儿。
需要培养习惯：去书店翻感兴趣的书；选自己当前需要的，感兴趣的，能读的进去的。不要开始参考别人的书单。一开始读太难的，容易消化不良。这时候，承认自己比较弱。一开始不要去追求高度和深度。不要求快，开始阶段，只是培养阅读习惯。读多了，自然而然有更高的要求。
刚开始，很容易找不到方向，读多了就会发现，有些领域是我想要深入钻研的，有些领域我能发光法热，变得很牛逼的
建议：在读书方面比较弱的时候，不要选择读小说，不要过于相信自己的自控力，读小说很容易陷入故事情节里面，很难实现个人成长。
刚开始阶段：多去逛书店；选自己读的进去的，且有用的书；大量的去读

主题阅读

读了大量书，养成读书习惯后，一天不读书内心痒，发现某个领域，买相关领域比较好的书，不要选读不懂的&特别深的书暂时放下， 这段时候基础仍然有点儿弱、很弱。买到主题的书，一本一本挨着读、挨着翻。
在知道足够多，也请教了很多达人&高手（活书）后，经过大量的练习，将自己的储备进行输出。输出后倒逼自己有更多的输入。输出的前提需要更多的优质输入。
读书：是个过程，需要做很多很多的事儿。需要大量的储备，产生输出。通过输出，倒逼大量的优质输入。
选书+读书（书＋人（活书））+输出（写+分享）+实践+复盘（总结反思）
读书，不仅仅是电子书、纸质书，还有人，向更专业的人沟通学习。
输出，会促使自己学的更好，而且输出会产生了更大的价值-利他，利他是最好的利己，给别人分享价值，提供价值。
实践：很多听课党（有课就会听），花了大量的时间被动输入，自己不去输出、实践，自然而然自己很难有更好的改变。
复盘：哪些做得好-可以持续，哪些做的不好-需要避免，养成复盘习惯，可以使实践的效果更好。

读书营的玩儿法

重头戏：五个主题营

第一个营 3月份开营 10天10本书
要求：快速翻完，找出对自己特别用用的点
提升：提升自己的选书能力，挑重点能力
第二个营 5月份开营 10天只读一本书
要求：10天读一本书，输出讲20~30min的逐字稿，优质者有机会分享
提升：提升输出能力
第三个营 7月份开营 10天共读一本书
要求：共读一本指定书籍，每个人的解读不一样，选出优胜者（读得好、总结好）进行分享
提升：联机学习，知道高手的能力
第四个营 9月份开营 10天10个故事
提升：讲故事能力
故作高深的人讲道理，真正高深的人讲故事
第五个营 11月份开营 大招（你知道的，我们总会给人惊喜）
总体：提升读写说能力，提升思考力
注意：
激励是有限的，读很多的书，记住的只是很少的一部分，剩下的是能力

每日精进

读书营设计内容：我们跟着参与就可以
争取每天进步一点点。

分享

有特别的礼物：勋章、服装、书籍

线下分享机会，现场奖杯，共进晚餐分享&交流

社群的真正价值是什么？

社群、分享、多交朋友（靠谱的，优质的）、让自己
朋友圈的高度决定了自己的高度
社群六件事儿： 读好书、见牛人、勤写作、爱分享、重践行、常利他
